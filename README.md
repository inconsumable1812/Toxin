# Toxin

Главная страница - https://inconsumable1812.github.io/Toxin/

## Используемые версии

Версия node `16.13.0`

Версия npm `8.1.0`

## Команды проекта

Установка зависимостей

`npm i`

Запуск dev-сервера

`npm start`

Сборка проекта

`npm run build`

Сборка проекта в режиме разработки

`npm run dev`

Запуск ESlint

`npm run lint`
