# Toxin

Главная страница - https://inconsumable1812.github.io/Toxin/

## Используемые версии

Версия node `14.17.3`

Версия npm `6.14.13`

## Команды проекта

Установка зависимостей

`npm i`

Запуск dev-сервера

`npm start`

Сборка проекта

`npm run build`

Сборка проекта в режиме разработки

`npm run dev`

Запуск ESlint

`npm run lint`
